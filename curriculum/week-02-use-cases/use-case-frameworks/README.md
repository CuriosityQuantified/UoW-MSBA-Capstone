# Use Case Frameworks & Grading

Frameworks and rubrics for evaluating and selecting use cases.

## Contents

This folder contains:
- Evaluation frameworks for narrowing down use cases
- Grading rubrics with scoring criteria
- Comparison matrices
- Decision tools

## Key Components of a Good Framework

1. **Problem Severity** — How painful is this problem?
2. **Data Availability** — Do you have (or can you get) the data needed?
3. **Technical Feasibility** — Can this be built in the time available?
4. **Team Expertise** — Do you understand the domain?
5. **Competitive Landscape** — Who else is solving this?
6. **User Validation** — Have you talked to actual users?
7. **Success Metrics** — How will you know you solved it?
