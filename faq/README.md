## How to Use This

Each file covers a cluster of related questions. Questions are written as a student would actually ask them — not as formal documentation.

If you're a contributor adding new material: when a student asks a question that would be useful to others, add the answer to the relevant file here. If no file fits, create a new one and add a row to the table above.
